#  Témoignages Codeloccol

Un mini-projet web qui présente les **témoignages des étudiants de Codeloccol**, avec un design moderne et responsive.
Chaque carte affiche la photo, le nom, le rôle et le témoignage d’un(e) étudiant(e).

---

## Objectif du projet

Mettre en pratique les bases de :

* **HTML5** : structure sémantique du contenu
* **CSS3** : mise en forme, dégradés, bordures stylisées
* **Flexbox & Grid** : disposition des cartes
* **Responsivité** : adaptation fluide à tous les écrans (desktop → mobile)


##  Technologies utilisées

* **HTML5**
* **CSS3**
* **Font Awesome** (pour les icônes d’étoiles)
